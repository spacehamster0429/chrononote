# 제3자 소프트웨어 고지

ChronoNote는 다음 브라우저용 제3자 소프트웨어를 원본 라이선스에 따라 배포합니다.

- DOMPurify 3.4.11 — Apache License 2.0 또는 Mozilla Public License 2.0. ChronoNote 배포본은 Apache License 2.0 조건을 따릅니다. 라이선스 원문: `public/vendor/dompurify/LICENSE`
- KaTeX 0.16.47 — MIT License. Copyright (c) 2013-2020 Khan Academy and other contributors. 라이선스 원문: `public/vendor/katex/LICENSE`
- Marked 18.0.5 — MIT License와 포함된 Markdown 구성요소의 BSD 조건. Copyright (c) MarkedJS, Christopher Jeffrey, John Gruber. 라이선스 원문: `public/vendor/marked/LICENSE`

서버 의존성은 `package-lock.json`에 버전과 라이선스 메타데이터가 기록되며, 설치된 각 npm 패키지는 자체 라이선스 조건을 유지합니다. ChronoNote의 Apache License 2.0은 제3자 소프트웨어의 라이선스를 대체하지 않습니다.
