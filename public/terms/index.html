이용약관

버전 2026-07-06 · 시행일 및 최종 수정: 2026년 7월 6일

1. 약관 동의와 이용 연령

ChronoNote에 회원가입하거나 서비스를 이용하면 본 약관과 개인정보 처리방침이 적용됩니다. 회원가입은 만 14세 이상만 할 수 있습니다. 약관에 동의하지 않거나 만 14세 미만인 경우에는 계정을 만들지 말고 비로그인 체험 모드만 이용해 주세요.

2. 제공 서비스

ChronoNote는 Markdown 메모 작성, 자동 복구본, 파일 단위 Git 기록, 첨부파일, 검색, ZIP 내보내기와 이용자가 직접 설정하는 GitHub 백업 기능을 제공합니다. 서비스는 개인이 무료로 운영하는 프로젝트이며 유료 결제나 광고를 제공하지 않습니다.

3. 계정과 보안

  • 이용자는 본인이 사용할 수 있는 이메일 또는 Google 계정으로 가입해야 합니다.
  • 비밀번호와 로그인 세션을 안전하게 관리하고, 무단 이용을 발견하면 세션을 종료해야 합니다.
  • 허위 정보, 타인 사칭, 계정 양도 또는 다른 이용자의 작업공간 접근을 시도해서는 안 됩니다.
  • 운영자는 악용·보안 사고·법령 위반을 막기 위해 계정 또는 기능 이용을 제한할 수 있습니다.

회원탈퇴 요청 후 48시간 동안 복구할 수 있으며, 이 기간이 지나면 작업공간을 삭제합니다.

4. 이용자 콘텐츠와 백업

메모와 첨부파일의 권리는 이용자에게 남습니다. 이용자는 자신이 작성·업로드할 권리가 있는 콘텐츠만 저장해야 합니다. 서비스 제공에 필요한 저장, 변환, 검색 인덱싱, 화면 표시와 이용자가 요청한 내보내기·GitHub 동기화 범위에서만 콘텐츠를 처리합니다.

Git 기록과 자동 복구 기능이 있더라도 영구 보존을 보장하지 않습니다. 중요한 자료는 ZIP 내보내기 또는 이용자 소유의 GitHub 저장소 등 별도 위치에도 백업해 주세요.

5. 금지 행위

다음 행위를 금지합니다.

  • 자동화된 대량 가입·요청, 용량 제한 우회 또는 서비스 운영 방해
  • 취약점 악용, 권한 우회, 타인의 데이터 또는 계정에 대한 무단 접근
  • 불법 콘텐츠, 악성코드, 스팸 또는 제3자의 권리를 침해하는 자료의 저장·배포
  • 서비스나 외부 API에 비정상적인 부하를 주는 행위

6. 외부 서비스

선택 기능은 Google OAuth와 GitHub에 의존합니다. 외부 서비스의 장애·정책 변경·계정 제한은 운영자가 통제할 수 없으며, 이용자는 해당 서비스의 약관도 준수해야 합니다. GitHub 동기화를 설정한 경우 저장소 공개 범위와 토큰 권한은 이용자가 직접 관리합니다.

7. 변경과 가용성

유지보수, 장애, 보안 대응, 외부 API 변경 또는 개인 운영 사정으로 기능을 변경하거나 서비스를 일시·영구 중단할 수 있습니다. 항상 이용 가능하거나 모든 데이터가 무기한 보존됨을 보장하지 않으며, 가능한 경우 중요한 변경과 장기 중단을 미리 안내합니다.

8. 책임 범위

운영자는 합리적인 범위에서 서비스를 안전하고 안정적으로 제공하려 노력합니다. 다만 운영자의 고의 또는 중대한 과실이 없는 한 이용자의 관리 소홀, 외부 서비스, 불가항력 또는 예측하기 어려운 장애로 발생한 간접 손해에 대한 책임은 제한됩니다. 관련 법령상 제한할 수 없는 책임은 제외됩니다.

9. 약관 변경

기능과 운영 방식의 변경을 반영하기 위해 약관을 수정할 수 있습니다. 변경 시 문서 상단의 버전과 최종 수정일을 갱신하고, 이용자 권리에 중요한 변경은 서비스 안에서 알리거나 다시 동의를 받습니다.

10. 준거와 분쟁 해결

본 약관은 대한민국 법령에 따릅니다. 분쟁이 발생하면 운영자와 이용자는 먼저 원만한 해결을 위해 협의하며, 해결되지 않는 경우 관련 법령이 정한 절차와 관할에 따릅니다.
